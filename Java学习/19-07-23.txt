this关键字：必须要是实例方法，才含有this，表示这个，static方法是没有实例的

在构造方法中使用this可以使用其他类型的构造方法
	例如：Data类中有两个构造函数，无参构造函数Data()，有参构造函数Data(int a),则在Data()中可以使用this(1)构造
		，而不产生新的对象

带有static的方法，既可以使用类名访问，也可以使用引用访问，不会有语法错误，但是尽量使用类名访问与实例方法进行区别

所有对象都有的属性：static变量
每个对象都不同的属性：实例变量

静态代码块：
	static {
		java语句；
	}
静态代码在类加载时执行，并且只执行一次，一个类中可以编写多个，自上而下执行
静态代码块的作用？怎么用？用在哪？
	例如：项目开发时的日志


java的继承： extends 父类名
java的继承只支持单继承

java中一个类没有显示继承任何类，该类默认继承javase库当中提供的java.lang.Object类

myeclipse：查找类快捷键：ctrl+shift+T
	查找资源快捷键：ctrl+shift+R

java中的方法覆盖：子类中对父类中的方法重新编写，访问权限不能更低

多态：
	向上转型：子类型-》父类型，自动类型转换
	向下转型：父类型-》子类型，强制类型转换，需要加强制类型转换符，当想要调用子类独有的方法的时候，可以使用
		instanceof运算符可以避免向下转型的时候出现异常
		语法：（引用 instanceof 类型名），返回boolean类型
		在强制类型转换之前，建议用instanceof进行判断
	编译阶段与运行阶段绑定状态不同
作用！！！降低程序的耦合度，提高程序的扩展力，父类型引用子类型；
	
final关键字：表示最终的，不可变的，类无法继承，方法无法覆盖，变量赋值后不可修改，
